# UI icons

Toolbar, sidebar, and other in-app icons. Wireframe style (stroke, no fill), 24×24 viewBox. Bundled via `icons.qrc`; use resource paths `:/icons/cog.svg`, `:/icons/quill.svg` (e.g. `QIcon(":/icons/cog.svg")`).

- **cog.svg**, **quill.svg** — from [Plume](../../plume) `ui/icons/` (settings, compose).
- **reply.svg**, **reply-all.svg**, **forward.svg** — message actions (from [Lucide](../../lucide/icons)).
- **junk.svg** — ban/no symbol for Mark as junk.
- **move.svg** — folder with arrow for Move (button hidden; move via drag-drop from message list to folder pane).
- **trash.svg** — trash can for Delete.
