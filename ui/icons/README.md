# UI icons

Toolbar, sidebar, and other in-app icons. Wireframe style (stroke, no fill), 24×24 viewBox. Bundled via `icons.qrc`; use resource paths `:/icons/cog.svg`, `:/icons/quill.svg` (e.g. `QIcon(":/icons/cog.svg")`).

- **cog.svg**, **quill.svg** — from [Plume](../../plume) `ui/icons/` (settings, compose).
